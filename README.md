# Employee Payroll Management System

An intermediate-level Java console application built to manage employee records, compute dynamic payroll breakdowns, and view analytics reports powered by a backend MySQL Database.

## Features
- **Employee Registration:** Secure CRUD-like operations using JDBC.
- **Multithreaded Processing:** Uses separate worker threads to process salary details concurrently.
- **Analytical Reporting:** Generates text-based operational layout summaries.

## Technologies Used
- Java SE (Object-Oriented Architecture, Multithreading)
- JDBC (Java Database Connectivity)
- MySQL / MariaDB Server
